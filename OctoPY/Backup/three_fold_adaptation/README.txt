- Connect to ThymioNet with fixed IP address
- Change EXPERIMENT_NAME and everything necessary in parameters.py
- cd into the /scripts folder
- Run "./copy_file.sh parameters.py" without quotes on a shell
- Run "./reboot_all.sh" without quotes on a shell
- Wait for 1 minute
- Make sure that the RPis are ready by running "./ping_all.sh" without quotes on a shell
- When they are all ready run "./start_all.sh -d" without quotes on a shell
- Once it's done, check that every file has been received correctly under /received_outputs folder
- If not, repeat the experiment
